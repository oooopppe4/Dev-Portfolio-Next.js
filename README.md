## üíª Run Locally

Once you have customized your portfolio, it's time to see it in action. Follow these steps:

1. **Open Your Terminal**
   Launch the terminal or command prompt on your computer.

2. **Navigate to the Project Directory**
   Use the `cd` command to go to the folder where you extracted the application files.

3. **Install Dependencies**
   If you have not installed the required tools through the command line interface (CLI), enter:
   ```bash
   npm install
   ```

4. **Start the Application**
   Run the following command to start the development server:
   ```bash
   npm run dev
   ```
   This command will launch your portfolio locally. Open your web browser and enter `http://localhost:3000` to view your changes.
